#### Generation

#### Characteristics
```c++
constexpr result_type min() noexcept;

```

### Notes
`#!c++ pxart::mt19937` used with `#!c++ pxart::mt19937::default_seeder` produces exactly the same results as `#!c++ std::mt19937` from the C++ STL.
